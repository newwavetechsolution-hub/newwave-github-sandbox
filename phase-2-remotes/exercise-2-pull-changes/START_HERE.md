> **Note:** All Git commands work the same on Mac/Linux/Windows (Git Bash/PowerShell/CMD).

1. [ ] Ensure you're on `main`: `git checkout main`

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git checkout main
   ```

2. [ ] Pull: `git pull origin main`

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git pull origin main
   ```

3. [ ] Switch to your branch (e.g. `feature/my-first-push`). Merge main in: `git merge main`

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git checkout feature/my-first-push
   git merge main
   ```

4. [ ] If there are no conflicts, push your branch again (or skip if already up to date)

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git push origin feature/my-first-push
   ```
