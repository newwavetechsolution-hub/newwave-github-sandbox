# START HERE — Merge branch

> **Note:** All Git commands work the same on Mac/Linux/Windows (Git Bash/PowerShell/CMD).

1. [ ] Switch to main: `git checkout main`

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git checkout main
   ```

2. [ ] Merge your feature branch: `git merge feature/your-name` (use the branch you created in exercise-1)

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git merge feature/your-name
   ```

3. [ ] Run `git log --oneline --graph` — you should see the merge (or fast-forward)

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git log --oneline --graph
   ```

4. [ ] Push main if you use a remote: `git push origin main`

   **Mac/Linux/Windows (Git Bash/PowerShell/CMD):**
   ```bash
   git push origin main
   ```
